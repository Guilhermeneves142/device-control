# Design system

Este grupo documenta os **tokens de design** usados na app (`src/styles/tokens/` + `src/style.css`), visíveis no Storybook graças ao import global em `.storybook/preview.ts`.

## Páginas

### Foundations
Base do design system: cores, tipografia, espaçamentos e tokens visuais.

**Cores**: Paleta semântica (primárias, feedback, texto, superfície)              |

**Tipografia**: Tamanhos, pesos, interlineado e família                                |

**Espaçamentos**: Escala `--spacing-*` e exemplo de `gap`                              |

**Radius**: `border-radius` (`--radius-*`)                                          |

## Boas práticas

- Preferir sempre **variáveis CSS** (`var(--token)`) em vez de valores fixos repetidos.
- Ao adicionar um token novo, atualizar o ficheiro em `src/styles/tokens/` **e** a página MDX correspondente aqui no Storybook.
